# Database

---

## CREATE 建立資料庫

```SQL
CREATE DATABASE databasename;

-- 建立資料庫，並指定其預設的字符集
CREATE DATABASE dbname CHARACTER SET utf8 COLLATE utf8_general_ci;
```

## USE 指定使用的資料庫

```SQL
USE databasename;
```

## SHOW 查看包含多少資料庫

```SQL
SHOW DATABASES;
```

## DROP 刪除資料庫

```SQL
DROP DATABASE databasename;
```
